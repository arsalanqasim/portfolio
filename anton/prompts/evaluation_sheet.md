# Anton Character Concept Evaluation Sheet

Every candidate concept art file must be scored and reviewed using this matrix before being moved to the `approved` directory.

---

## 1. Evaluation Scoring Matrix (Scale 1–10)

| Metric | Criteria Description | Target Focus | Score (1-10) |
| :--- | :--- | :--- | :---: |
| **Silhouette** | Distinctiveness and legibility at small sizes (e.g. 55px on nav bar). | Can you recognize it instantly? | |
| **Intelligence** | Eye styling and posture convey a thoughtful observer. | Does it look like a thinker, not a pet? | |
| **Curiosity** | Overall head angle and focus target alignment. | Does it feel genuinely inquisitive? | |
| **Timelessness** | Minimalist aesthetic, avoids trends, colors remain modern. | Will this design work 10 years from now? | |
| **Brand Recognition** | Fits the aesthetic tone of a premium calisthenics & engineering brand. | Does it feel professional, clean, organic? | |
| **Portfolio Fit** | Complements the dark, sleek website themes (Linear/Figma style). | Does it sit naturally on container card borders?| |
| **Animation Potential**| Joints, eyelids, limbs, and tail shapes are rigged easily. | Can it splay, scurry, climb, and freeze? | |
| **Overall** | Consolidated score representing readiness for v1.0. | Recommended for development? | |

---

## 2. Review Verdict Status
Check one of the following statuses for the concept:
- [ ] **Keep** (Promote to active WIP or approved index)
- [ ] **Reject** (Move to archive/rejected directory with explanation)
- [ ] **Iterate** (Feed back to prompt modifications with iteration notes)
- [ ] **Lock Candidate** (Ready for production final lock)
